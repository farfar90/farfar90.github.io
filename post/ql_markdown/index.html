快速使用：MarkDown

Jun 07, 2022
1 minute read

MarkDown元素分类

MarkDown元素分为两类，块级元素和行内元素。块级元素至少占据一行，也可以占据多行；行内元素只占据自身所需要的空间，不满一行，后面还可以有其他元素。

块级元素

  • 标题：

    • 描述：标题分为六级，使用#号标记，依次增加个数，最多六个
    • 使用：# [标题]
    • 演示效果：待更新

  • 引用：

    • 描述：引用时使用
    • 使用：> [文本]
    • 演示效果：待更新

  • 列表：

    • 有序列表：

      • 描述：使用数字为标识的列表，数字加句号,再加上空格
      • 使用：1. 有序列表
      • 演示效果：待更新

    • 无序列表：

      • 描述：使用-号标记，也可以使用*号标记，加空格；无序列表一共可以分成三级，使用Tab键缩进以改变级数
      • 使用：- 无序列表
      • 演示效果：待更新

  • 任务列表：

    • 描述：带有可选择项的列表，可以勾选，可以在缩进里使用

    • 使用：

      • 未勾选：-[ ]
      • 勾选：-[x]

    • 演示效果：待更新

  • 代码块：

    • 描述：可以保留源代码风格
    • 使用：以三个`开始，可以加上程序语言名称，指定快内容对应的程序语言，以```结束
    • 演示效果：

java

int a = 6;

int a = 6;

  • 数学公式：

    • 描述：书写公式时使用
    • 使用：以$$开始，以$$结束
    • 问题：hugo上使用存在问题，待解决

  • 表格：

    • 描述：用于快速制作表格
    • 使用：
      • 表格表头：|姓名|性别|年龄|
      • 表格第二行：|:-|-:|:-:|，冒号规定对齐方式
      • 表格内容：|张三|男|18|，按照表头依次填写即可
    • 演示效果：待更新

  • 脚注：

    • 描述：为内容添加脚注，可以实现跳转到文末解释，可用于参考文献或解释
    • 使用：
      • 文内：使用[]包括，内容以^起头，内容可填写为解释的词语，[^脚注]
      • 脚注解释：[^脚注]:"解释内容"
    • 演示效果：脚注[^脚注]

  • 分割线：

    • 描述：分割文章内容
    • 使用：---
    • 使用效果：

行内元素

  • 斜体：

    • 使用：以前后各一个*号包括需要斜体的内容
    • 快捷方式：ctrl+i
    • 演示效果：斜体

  • 加粗：

    • 使用：前后各两个*号包括需要加粗的内容
    • 快捷按键：ctrl + b
    • 演示效果：加粗

  • 行内代码块：

    • 描述：可以保留源代码风格
    • 使用：以一个`开头和结尾，中间填充内容
    • 演示效果：print()

  • 下划线：

    • 使用：<h>内容</h>
    • 快捷按钮：ctrl + u
    • 演示效果：下划线

  • 国际通用表情：

    • 使用：:smile:
    • 演示效果：:smile:
    • 可参考：
      • 表情符号简码列表

  • 链接：

    • 使用：[文本内容](链接[空格分隔]"鼠标悬停显示的标题")
    • 演示效果：百度

  • 引用链接：

    • 描述：同个链接多次使用
    • 使用：
      • 文内：[百度][id]
      • 链接描述：[id]:http://www.baidu.com "一个搜索引擎"，不在文内显示
      • 演示效果：百度
  • 链接到标题：
    • 描述：可以实现点击跳转
    • 使用：暂时没弄懂
    • 演示效果：无
  • URL:
    • 描述：使用网址时，以http或https头，可直接显示为链接，在文内时用空格分隔
    • 演示效果： http://www.baidu.com
  • 图片：
    • 描述：可以添加图片
    • 使用：![替代文本](图片链接 "可选，图片标题文本")
    • 问题：hugo中怎么使用图片，markdown文档内的效果是使用href进行跳转的
